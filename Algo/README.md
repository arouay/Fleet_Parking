# FizzBuzz - Algorithm Test

## Description
This project provides a clean and testable implementation of the classic FizzBuzz problem.

The exercise rules are described in the `fizzbuzz.md` file at the root of this repository.

## Features
- Pure function implementation
- Separation between business logic and CLI execution
- Unit tests for core logic
- Integration tests for CLI behavior

## Requirements
- Node.js >= 14

## Installation
```bash
git clone <your-repo-url>
cd <your-repo>
npm install
```

## Usage

Run the script from the command line:

```bash
node fizzbuzz.js <maxNumber>
```

## Input validation
```markdown
- The input must be a positive integer
- Zero, negative numbers, and non-numeric values will return an error
```
